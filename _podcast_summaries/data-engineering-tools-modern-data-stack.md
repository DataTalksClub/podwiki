# ETL vs ELT & Data Lake vs Warehouse: Airbyte, dbt, CDC for Modern Data Engineering

## Why This Episode Matters

This is a foundational modern data stack explainer. It covers ETL/ELT, ingestion, transformations, analytics engineering, data lakes, warehouses, orchestration, reverse data flows, open source, CDC, and schema evolution. It is useful when future agents need concise archive evidence for data platform terminology.
